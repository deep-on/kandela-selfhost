# Kandela 베타 서비스 이용 안내 및 면책 고지

**버전: 1.0 | 시행일: 2026-03-18 | 적용 범위: 베타 기간 전체**

---

## 베타 서비스란?

Kandela 베타는 정식 출시 전 소규모 사용자를 대상으로 제공하는 **테스트 목적의 서비스**입니다.
기능, 안정성, 데이터 보존 측면에서 정식 서비스와 다를 수 있으며, 아래 사항을 충분히 이해하신 후 이용해 주세요.

---

## 1. 데이터 보장 없음

- 베타 기간 중 서버 초기화, 데이터베이스 마이그레이션, 구조 변경 등으로 **저장된 기억(메모리) 데이터가 예고 없이 삭제될 수 있습니다.**
- 회사는 베타 기간 중 데이터 손실에 대해 **책임을 지지 않습니다.**
- 중요한 정보는 반드시 별도로 백업해 두시기 바랍니다.

## 2. 서비스 중단 가능성

- 베타 서비스는 **사전 예고 없이 일시 중단 또는 종료**될 수 있습니다.
- 서비스 가용성(uptime) SLA를 보장하지 않습니다.
- 정식 출시 전 베타 종료 시, 데이터 내보내기(export) 기회를 7일 이상 제공하기 위해 노력합니다.

## 3. 기능 변경 가능성

- 베타 기간 중 기능, API, 설치 스크립트, 훅 동작 방식이 **사전 예고 없이 변경**될 수 있습니다.
- 변경으로 인해 클라이언트 재설치가 필요할 수 있습니다.
- 주요 변경 사항은 이메일 또는 대시보드를 통해 안내합니다.

## 4. 보안 및 프라이버시

- 저장되는 데이터(프로젝트 기억, 세션 요약, 코드 스니펫 등)는 암호화되어 저장됩니다.
- 단, 베타 기간 중 **보안 취약점이 발견될 수 있으며**, 발견 즉시 패치합니다.
- **민감한 개인정보, 회사 기밀, 인증 정보(패스워드, API 키 등)는 저장하지 마세요.**
- 개인정보 처리 방침: [kandela.ai/privacy](https://kandela.ai/privacy)

## 5. 로컬 설치 관련

- 설치 스크립트는 `~/.claude/settings.json` 및 `~/.claude/hooks/` 디렉토리를 수정합니다.
- 설치 전 상태로 완전히 복구하는 언인스톨 기능을 제공합니다.
- 단, **Claude Code 버전 변경, OS 업데이트 등 외부 변화로 인한 충돌**에 대해서는 책임을 지지 않습니다.

## 6. 무료 제공 및 요금 변경

- 베타 기간 중 서비스는 **무료로 제공**됩니다.
- 정식 출시 시 유료 전환될 수 있으며, **전환 30일 전 이메일로 안내**합니다.
- 유료 전환을 원하지 않으면 전환 전에 자유롭게 탈퇴할 수 있습니다.

## 7. 면책 범위

회사는 다음에 대해 책임을 지지 않습니다:

- 베타 서비스 이용 중 발생한 **데이터 손실, 업무 손해, 기회 손실**
- 서비스 중단으로 인한 **간접 손해**
- Claude Code, Anthropic 서비스 등 **제3자 플랫폼의 변경 또는 장애**로 인한 문제
- 사용자가 저장한 정보의 **정확성, 완전성**

## 8. 베타 종료 및 데이터 처리

| 상황 | 처리 방식 |
|------|----------|
| 베타 → 정식 전환 | 데이터 자동 이관, 별도 재설치 불필요 |
| 베타 종료 (정식 미출시) | 종료 30일 전 안내 + 데이터 내보내기 기회 제공 후 전체 삭제 |
| 사용자 직접 탈퇴 | 즉시 데이터 삭제 (요청 시) 또는 30일 후 자동 삭제 |
