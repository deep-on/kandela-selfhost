# 이용약관

**시행일: 2026년 3월 15일**
**버전: 1.0**

---

## 제1조 (목적)

이 약관은 딥온 주식회사(이하 "회사")가 제공하는 Memory MCP Server 서비스(이하 "서비스")의 이용 조건 및 절차, 회사와 이용자 간의 권리, 의무 및 책임사항, 기타 필요한 사항을 규정함을 목적으로 합니다.

---

## 제2조 (용어의 정의)

이 약관에서 사용하는 주요 용어의 정의는 다음과 같습니다.

| 용어 | 정의 |
|------|------|
| **서비스** | 회사가 제공하는 Memory MCP Server, 이에 부수하는 웹 대시보드, API, 텔레그램 봇 등 일체의 서비스 |
| **이용자** | 이 약관에 따라 서비스에 가입하여 회사가 제공하는 서비스를 이용하는 자 |
| **메모리** | 이용자가 서비스를 통해 저장하는 텍스트 콘텐츠(결정사항, 코드 스니펫, 요약, 사실 정보 등) |
| **프로젝트** | 이용자가 메모리를 분류하기 위해 생성하는 논리적 단위 |
| **API 키** | 서비스 인증을 위해 발급되는 고유 식별 문자열 |
| **MCP** | Model Context Protocol. AI 코딩 도구와 서비스 간의 통신 프로토콜 |
| **대시보드** | 서비스의 웹 기반 관리 인터페이스 |

이 약관에서 정의하지 않은 용어는 관계 법령 및 일반적인 상관례에 따릅니다.

---

## 제3조 (약관의 효력 및 변경)

1. 이 약관은 서비스 화면에 게시하거나 기타의 방법으로 이용자에게 공지함으로써 효력이 발생합니다.
2. 회사는 합리적인 사유가 발생할 경우 관계 법령에 위배되지 않는 범위에서 이 약관을 변경할 수 있습니다.
3. 약관이 변경되는 경우 회사는 변경 사항을 시행일 7일 전부터 서비스 대시보드 또는 이메일을 통해 공지합니다. 다만, 이용자에게 불리하거나 중대한 변경의 경우 30일 전에 공지합니다.
4. 변경된 약관에 동의하지 않는 이용자는 서비스 이용을 중단하고 탈퇴할 수 있습니다. 약관 변경 공지 후 변경 시행일까지 거부 의사를 표시하지 않은 경우 변경된 약관에 동의한 것으로 간주합니다.

---

## 제4조 (서비스의 내용)

1. 회사가 제공하는 서비스는 다음과 같습니다.

| 서비스 | 설명 |
|--------|------|
| 메모리 저장/검색/관리 | MCP 프로토콜을 통한 텍스트 메모리의 저장, 시맨틱 검색, 수정, 삭제 |
| 웹 대시보드 | 메모리 조회, 검색, 통계, API 키 관리, 계정 관리 |
| 자동 기억 관리 | 세션 시작/종료 시 자동 회상 및 요약, 로컬 캐시 동기화 |
| 텔레그램 봇 | 텔레그램을 통한 메모리 저장/검색 (제공 여부는 이용 등급에 따름) |

2. 서비스는 AI 코딩 도구(Claude Code, Cursor 등)의 보조 도구로서 메모리 저장 및 검색 기능을 제공하며, AI 시스템 자체가 아닙니다.
3. 회사는 Anthropic, OpenAI 등 AI 서비스 제공 업체와 제휴 또는 종속 관계가 없는 독립적인 서비스입니다.

---

## 제5조 (회원가입 및 탈퇴)

### 1. 회원가입

1. 이용자는 회사가 정한 가입 양식에 따라 필요한 정보를 입력하고, 이 약관과 개인정보처리방침에 동의함으로써 회원가입을 신청합니다.
2. 베타 서비스 기간 중 회원가입은 초대 코드(Invite Code)를 보유한 이용자에 한하여 허용됩니다.
3. 회사는 다음 각 호에 해당하는 신청에 대하여 가입을 거절하거나 사후에 이용 계약을 해지할 수 있습니다.
   - 타인의 정보를 도용한 경우
   - 가입 신청서의 내용을 허위로 기재한 경우
   - 이 약관에 따라 이전에 자격이 상실된 적이 있는 경우
   - 기타 가입을 승낙하는 것이 회사의 기술상 현저히 지장이 있는 경우
4. 이용자는 만 14세 이상이어야 합니다.

### 2. 회원 탈퇴

1. 이용자는 언제든지 대시보드의 계정 삭제(Delete Account) 기능 또는 이메일 요청을 통하여 탈퇴를 신청할 수 있습니다.
2. 탈퇴 시 이용자의 계정 정보, 저장된 메모리, API 키 등 모든 데이터가 삭제됩니다.
3. 탈퇴 처리 후 데이터 복구는 불가능합니다. 필요한 데이터는 탈퇴 전에 데이터 내보내기(Export) 기능을 이용하여 백업하시기 바랍니다.
4. 관계 법령에 따라 보존이 필요한 정보는 해당 법령에서 정한 기간 동안 보관됩니다.

---

## 제6조 (서비스 이용료)

1. **베타 서비스 기간**: 서비스의 기본 기능은 무료로 제공됩니다.
2. **유료 전환 시**: 회사가 유료 서비스(Pro 등급 등)를 도입하는 경우, 도입 30일 전에 이용자에게 서비스 내용, 이용료, 결제 방법 등을 사전 고지합니다.
3. 유료 전환 시 기존 무료 이용자에게는 합리적인 전환 기간을 제공하며, 유료 전환에 동의하지 않는 이용자는 서비스를 계속 이용하거나 탈퇴할 수 있습니다.
4. 유료 서비스 도입 시 별도의 이용 약관 및 환불 정책을 수립하여 공지합니다.

---

## 제7조 (서비스의 변경 및 중단)

1. 회사는 서비스의 내용, 운영 방식 등을 변경할 수 있으며, 변경 사항은 서비스 대시보드 또는 이메일을 통해 사전에 공지합니다.
2. 회사는 다음 각 호에 해당하는 경우 서비스의 전부 또는 일부를 제한하거나 중단할 수 있습니다.
   - 서비스용 설비의 보수, 점검, 교체, 고장, 통신 두절 등의 사유가 발생한 경우
   - 천재지변, 국가비상사태, 전력 공급 중단 등 불가항력의 사유가 발생한 경우
   - 서비스 이용의 폭주 등으로 정상적인 서비스 이용에 지장이 있는 경우
3. 서비스를 종료하는 경우 회사는 종료일로부터 최소 30일 전에 이용자에게 통지하며, 이용자가 데이터를 내보낼 수 있는 충분한 기간을 제공합니다.
4. 베타 서비스 기간 중에는 사전 공지 없이 서비스가 일시적으로 중단될 수 있으며, 이에 따른 손해에 대해 회사는 책임을 부담하지 않습니다.

---

## 제8조 (이용자의 의무 및 금지 행위)

1. 이용자는 다음 행위를 하여서는 안 됩니다.

| 금지 행위 | 설명 |
|-----------|------|
| **불법 콘텐츠 저장** | 법령에 위반되는 내용, 불법 정보, 음란물 등의 저장 |
| **타인 개인정보 저장** | 제3자의 개인정보(주민등록번호, 카드번호, 비밀번호 등)를 이용자의 메모리에 저장하는 행위 |
| **시스템 공격** | 서비스의 안정적 운영을 방해하는 행위 (DDoS, 해킹, 무단 접근 시도 등) |
| **API 남용** | 비정상적으로 대량의 요청을 발생시키는 행위, API 키의 무단 공유 |
| **타인 계정 도용** | 타인의 계정, API 키를 무단으로 사용하는 행위 |
| **서비스 역공학** | 서비스 인프라(서버, 네트워크 등)에 대한 무단 접근, 보안 우회 시도. 단, 소프트웨어 소스 코드에 대한 열람·수정·배포는 해당 오픈소스 라이선스(AGPL-3.0)에 따라 허용됨 |
| **재판매** | 회사의 사전 서면 동의 없이 서비스를 재판매하거나 재배포하는 행위 |

2. 이용자는 관계 법령, 이 약관의 규정, 이용 안내 및 서비스와 관련하여 공지한 주의사항을 준수하여야 합니다.
3. 회사는 이용자가 본 조의 의무를 위반한 경우 서비스 이용을 제한하거나 계약을 해지할 수 있습니다.

### 제8조의2 (오픈소스 라이선스)

1. Kandela 서버 소프트웨어는 GNU Affero General Public License v3.0 (AGPL-3.0)에 따라 배포됩니다.
2. 클라이언트 소프트웨어(설치 스크립트, hooks, 슬래시 명령)는 MIT License에 따라 배포됩니다.
3. 오픈소스 라이선스에 따른 권리(소스 코드 열람, 수정, 재배포 등)는 본 약관의 어떤 조항에 의해서도 제한되지 않습니다.
4. 소스 코드 접근: support@kandela.ai로 요청하시거나, 공개 시 GitHub 저장소에서 확인할 수 있습니다.

## 제9조 (콘텐츠 소유권)

1. **이용자 콘텐츠의 소유권**: 이용자가 서비스를 통해 저장한 메모리 콘텐츠의 소유권은 해당 이용자에게 있습니다.
2. **서비스 운영을 위한 이용 허락**: 이용자는 서비스 제공에 필요한 범위 내에서(저장, 인덱싱, 검색, 표시 등) 회사가 해당 콘텐츠를 이용하는 것을 허락합니다.
3. **콘텐츠 활용 제한**: 회사는 이용자의 메모리 콘텐츠를 다음의 목적으로 사용하지 않습니다.
   - AI 모델 학습 또는 훈련
   - 광고 또는 마케팅
   - 서비스 제공 목적 외의 이용
   - 제3자에 대한 공개 또는 판매
4. **데이터 이동권**: 이용자는 언제든지 대시보드의 데이터 내보내기(Export) 기능을 통해 자신의 콘텐츠를 JSON 형식으로 내보낼 수 있습니다.

---

## 제10조 (면책조항)

1. **베타 서비스**: 본 서비스는 베타(Beta) 서비스로 제공되며, "있는 그대로(AS-IS)" 제공됩니다. 회사는 서비스의 완전성, 정확성, 신뢰성, 이용 가능성에 대해 명시적 또는 묵시적 보증을 하지 않습니다.
2. 회사는 다음 각 호의 사유로 인해 발생한 손해에 대해 책임을 부담하지 않습니다.
   - 천재지변, 전쟁, 테러, 폭동, 정부 조치, 통신 장애 등 불가항력에 의한 경우
   - 이용자의 귀책 사유로 인한 서비스 이용의 장애
   - 서비스의 변경, 중단, 종료로 인한 손해
   - 이용자가 저장한 메모리 콘텐츠의 정확성, 적법성에 관한 사항
   - 이용자 간 또는 이용자와 제3자 간의 분쟁
3. 회사는 이용자가 서비스를 통해 얻은 정보를 바탕으로 한 판단이나 행위에 대해 책임을 부담하지 않습니다.
4. 베타 서비스 기간 중 서비스 장애, 데이터 손실, 서비스 변경/종료 등이 발생할 수 있으며, 이에 대해 회사는 사전 예고 의무를 제외하고 별도의 보상을 하지 않습니다.

---

## 제11조 (데이터 백업 책임)

1. 이용자가 서비스를 통해 저장한 메모리 콘텐츠에 대한 백업은 이용자 본인의 책임입니다.
2. 회사는 서비스의 안정적 운영을 위해 합리적인 수준의 데이터 보호 조치를 취하고 있으나, 다음의 경우에 발생하는 데이터 손실에 대해서는 책임을 부담하지 않습니다.
   - 시스템 장애, 하드웨어 고장 등 기술적 문제
   - 불가항력에 의한 데이터 손실
   - 이용자의 실수 또는 부주의에 의한 데이터 삭제
3. 이용자는 대시보드의 데이터 내보내기(Export) 기능을 이용하여 정기적으로 데이터를 백업할 것을 권장합니다.

---

## 제12조 (손해배상)

1. 회사의 고의 또는 중대한 과실로 인하여 이용자에게 손해가 발생한 경우, 회사는 관계 법령에 따라 손해를 배상합니다.
2. 베타 서비스 기간 중 회사의 손해배상 책임은 해당 이용자가 서비스에 대해 지불한 금액을 한도로 합니다. 무료 베타 서비스의 경우 손해배상의 범위는 관련 법령이 허용하는 범위 내에서 최소한도로 제한됩니다.
3. 이용자가 이 약관을 위반하여 회사에 손해가 발생한 경우, 이용자는 그 손해를 배상하여야 합니다.

---

## 제13조 (분쟁 해결)

1. 이 약관과 관련하여 회사와 이용자 간에 발생한 분쟁에 대해서는 대한민국 법률을 준거법으로 합니다.
2. 서비스 이용과 관련하여 회사와 이용자 간에 분쟁이 발생한 경우, 양 당사자는 분쟁 해결을 위해 성실히 협의합니다.
3. 협의로 해결되지 않는 경우, **서울중앙지방법원**을 제1심 관할 법원으로 합니다.
4. 이용자는 개인정보와 관련한 분쟁의 경우 개인정보 분쟁조정위원회(kopico.go.kr)에 조정을 신청할 수 있습니다.

---

## 제14조 (기타)

1. 이 약관에서 정하지 아니한 사항과 이 약관의 해석에 관하여는 관계 법령 및 상관례에 따릅니다.
2. 이 약관의 일부 조항이 무효가 되더라도 나머지 조항의 효력에는 영향을 미치지 않습니다.
